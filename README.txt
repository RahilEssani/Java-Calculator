A mimic of windows 10 calculator using Java's swing library. The UI and calculating algo is 40% similar to windows 10. 
Doesn't contain the scientific features. A great UI with dynamic colors.

To Run This Project:
-Install Java JDK 8 on your system
-Create a java project in netbeans or elsewhere
-Copy all these files in that project (Copy .java files in src folder)
-Run Driver
